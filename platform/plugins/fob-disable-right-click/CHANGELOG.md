# Changelog

All notable changes to the FOB Disable Right Click plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-09

### Added
- Initial release
- Disable right-click context menu functionality
- Disable text selection functionality
- Disable developer console with auto-reload detection
- Keyboard shortcut blocking (F12, Ctrl+Shift+I, Ctrl+Shift+J, Ctrl+U, etc.)
- Settings page at `/admin/settings/fob-disable-right-click`
- Support for 42 languages:
  - Arabic, Bengali, Bulgarian, Chinese (Simplified), Chinese (Hong Kong)
  - Croatian, Czech, Danish, Dutch, English, Estonian, Filipino
  - Finnish, French, German, Greek, Hebrew, Hindi, Hungarian
  - Indonesian, Italian, Japanese, Georgian, Korean, Lithuanian, Latvian
  - Malay, Norwegian, Polish, Portuguese, Portuguese (Brazil)
  - Romanian, Russian, Serbian, Slovak, Slovenian, Spanish
  - Swedish, Thai, Turkish, Ukrainian, Vietnamese
- MIT License
- Comprehensive README documentation

### Security
- All protection mechanisms run client-side only
- No admin panel interference
- Proper permission checks for settings page
